\begin{center}\bfseries{Abstract}
\end{center}
\noindent\makebox[\textwidth][c]{
  \begin{minipage}{0.9\textwidth}
    In this thesis, we present the meta-telingo system, a temporal ASP
    system implemented using metaprogramming. Temporal ASP extends ASP
    with temporal modal operators, which act over a liner sequence of
    states. meta-telingo allows for a broader class of temporal
    programs than previous implementation of temporal ASP systems,
    such it's namesake, telingo, but operates only over a finite
    amount of states that must be set beforehand. The thesis contains
    an in-depth discussion of the theoretical underpinning of the
    meta-telingo system, with proofs of correctness, as well as
    including a discussion on the first-order semantics of temporal
    ASP program, tackling issues such as safety and efficient
    grounding. We also present another novel system, renopro, which
    allows users to apply metaprogramming techniques to non-ground
    clingo programs, a capability that did not exist beforehand.
  \end{minipage}}
